机器学习笔记：Clustering

发表于 2021-10-03 更新于 2021-10-20 分类于 Machine Learning

聚类是一种无监督学习算法。聚类的本质就是将相似的数据进行分组。主要方法包括： K-means，Mean-shift，Hierarchical cluster，SLICS, QUAD tree等等。

K-means

K-means方法通过一种方法确定K个中心点(centroids)进而形成K个群，然后将所有数据点分配到离它最近的群中。这里的中心点是数学意义上的均值点，不一定要求这个点在数据中存在。
算法的流程如下：

  • 随机选择K个中心点
  • 将每一个数据点分配到离它最近的的中心点所属的群。
  • 对于每一个群分别计算新的中心点。
  • 如果数据点的分配有变化，则重新执行步骤2，直到数据点的分配不再变化，此时得出的就是K个结果群。

Mean-shift

Mean-shift一种流行的图像分割技术，它可以在预先不设定分类数量的情况下对图片进行有效的分割。这种技术通过密度导向将导向同一点的数据分为同一类。
优点：

  • 具有很强的泛用性，支持几乎所有的图像数据，甚至包括一些非图像数据。
  • 不受分类数量和形状的限制，不用提前设定类数。
  • 对于异常点也可以完成分类。

缺点：

  • 必须事先设定窗口的大小，也就是说必须事先规定什么叫做“近”。
  • 对于高维数据不适用

算法流程如下：

  • 首先选定窗口函数和窗口大小
  • 对于每一个数据点：
    1. 计算以它为中心的窗口
    2. 计算窗口中数据的均值点
    3. 将窗口移植新的均值点

Hierarchical Clustering

Hierarchical Clustering是一种基于数据距离构建聚类层级结构的技术。它分为“自顶向下”和“自底向上”两大类。其中“自顶向下”指的是一开始将所有的数据认为是同一类，而后在不断的迭代中将数据进行分类，从而构建聚类层级结构。而“自底向上”是完全相反的情况。

Hierarchical Clustering通过两种函数确定数据间的距离和类间的距离。其中数据间的距离通过距离函数决定，常见的距离函数包括欧几里得距离(Euclidean distance)，曼哈顿距离(Manhattan distance)等， 在处理文本数据时，汉明距离(Hamming distance)也是常用的函数之一。类间距离的判断则通过最大连接(全连接)，最小链接(单链接)，平均链接等方式来确定。其中最大连接是指类间距离等于分属两类的数据点之间最大的距离。最小链接和平均链接可以以此类推。

nltk python3接口变更

发表于 2016-06-04 更新于 2016-11-24 分类于 Nltk

翻译自Nltk官方Wiki，原文链接点击

Nltk在python3的版本中更改了部分接口，并且现在Nltk要求所有的输入文件为unicode格式，并且也将只返回unicode格式的文件。之前有的类需要以bytestrings编码，现在你需要检查你的程序，因为bytestrings编码现在可能导致错误。

接口的改变：

  • grammar:ContextFreeGrammar->CFG,WeightGrammar->PCFG,
    StatisticalDependencyGrammar->ProbabilisticDependencyGrammar,
    WeightedProduction->ProbailisticProduction
  • draw.tree:TreeSegmentWidget.node()->TreeSegmentWidget.label(),
    TreeSegmentWidget.set_node()->TreeSegmentWidget.set_label()
  • parsers:nbset_parse()->parse()
  • ccg.parse.chart:EdgeI.next()->EdgeI.nextsym
  • Chunk parser:top_node->root_label,chunk_node->chunk_label
  • WordNet特性现在转变为方法，比如：Synset.definition->Synset.definition()
  • sem.relextract:mk_pairs()->_tree2semi_rel(),mk_reldicts()->semi_rel2reldict(),
    show_clause()->clause(),show_raw_rtuple()->rtuple()
